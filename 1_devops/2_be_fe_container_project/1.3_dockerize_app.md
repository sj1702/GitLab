
# dockerize app

- added Dockerfile as follows:
```text
FROM node:20.11.1
WORKDIR /displaysum/
COPY public/ /displaysum/public
COPY src/ /displaysum/src
COPY package.json /displaysum/
RUN npm install
RUN npm i bootstrap
CMD ["npm", "start"]
```

- build image
```text
docker image build -t displaysum:1.0 .
```

- run image
```text
docker run -dp 3000:3000 --name displaysum displaysum:1.0
```
